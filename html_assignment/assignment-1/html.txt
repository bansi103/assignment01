1. explain the structure of an html document and the purpose of DOCTYPE.
ANS:  <!DOCTYPE>: declration define that this document is an html5 document
      <html>: The <html> element is the root element of an HTML page.
      <head>: the <head> element contains of meta information of an html page.
      <body>: The <body> element defines the document's body, and is a container for all the visible contents.
      <title>: the <title> specific titel of the html.

      purpose of DOCTYPE:The primary purpose of the DOCTYPE declaration is to inform browsers about
